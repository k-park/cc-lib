# User Personas

**Version**: 1.0.0
**Last Updated**: 2026-01-11
**Status**: Draft

## Overview

cc-lib는 주로 **Claude Code를 사용하는 개발자들**을 타겟으로 합니다.

## Primary Personas

### Persona 1: 프로 개발자 (Alex)

| Attribute | Description |
|-----------|-------------|
| **Name** | Alex Kim |
| **Role** | Full-stack Developer |
| **Experience** | 5+ years |
| **Environment** | PC/Desktop (VS Code, IntelliJ) |
| **Goals** | 생산성 향상, 반복 작업 자동화 |
| **Frustrations** | 에이전트를 일일이 찾고 설정하는 것이 번거로움 |
| **Technical Skill** | High |

**Quote**: "나는 내 워크플로우에 맞는 에이전트를 빠르게 찾아서 설치하고 싶어."

**Use Cases**:
- 코드 리뷰 자동화
- 테스트 생성
- 문서 자동화

## Persona Summary

| Persona | Priority | Key Needs |
|---------|----------|-----------|
| 프로 개발자 (Alex) | High | 빠른 발견, 설치, 신뢰성 |
| 모바일 개발자 (Sam) | Medium | 경량화, 이동성 |
| DevOps 엔지니어 (Jordan) | Medium | 자동화, CI/CD 통합 |
| 주니어 개발자 (Taylor) | Medium | 학습 리소스, 가이드 |

## Anti-Personas

**Not targeting**:
- 비개발자 (경영진, PM, 디자이너 등)
- Claude Code를 사용하지 않는 사용자
- 완전한 노코드 사용자
