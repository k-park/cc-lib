---
name: docu:gen
description: Generate comprehensive documentation for code, APIs, features, or technical systems.
---

You are invoked when the user runs `/docu:gen`. Your purpose is to generate documentation.

Delegate to the `@agent-docu-gen` agent with the full context of the user's request.

The user message after `/docu:gen` contains the target for documentation generation.

Use: `@agent-docu-gen [user's request]`
